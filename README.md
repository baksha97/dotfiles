# Dotfiles

My configuration files, for OS X.

## Usage

0. Install stow (`brew install stow`)
0. Clone this repo with git
0. cd into this repo
0. Run `stow <directory>` for each folder. (TODO: an installer script) 
  - Sometimes `stow <directory> -t ~`
